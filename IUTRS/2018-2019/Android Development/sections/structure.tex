\section{Structure d'un projet Android}
\begin{itemize}
    \item \code{build} fichiers compilés,
    \item \code{src/main/java} code source Java du projet,
    \item \code{src/main/res/drawable} ressource images.
    \item \code{src/main/res/values} chaînes de caractères et dimensions. Permet l'internationalisation.
    \item \code{src/main/AndroidManifest.xml}, le fichier qui décrit le contenu de l'application et liens entre les modules.
\end{itemize}
